**Helper function (reusable for all workflows):**
```python
def _ensure_workflow_state(state_name, style=""):
    if not frappe.db.exists("Workflow State", state_name):
        frappe.get_doc({"doctype": "Workflow State", "workflow_state_name": state_name, "style": style}).insert(ignore_permissions=True)

def _ensure_workflow_action(action_name):
    if not frappe.db.exists("Workflow Action Master", action_name):
        frappe.get_doc({"doctype": "Workflow Action Master", "workflow_action_name": action_name}).insert(ignore_permissions=True)

def _create_workflow(name, document_type, states, transitions):
    # Check for existing active workflow on same doctype (pitfall 6)
    existing = frappe.get_all("Workflow", {"document_type": document_type, "is_active": 1})
    if existing:
        print(f"  Active workflow already exists for {document_type}: {existing[0].name}")
        return existing[0].name
    # Ensure all states and actions exist first (pitfall 2)
    for s in states:
        _ensure_workflow_state(s["state"], s.get("style", ""))
    for t in transitions:
        _ensure_workflow_action(t["action"])
    # Create workflow
    wf = frappe.get_doc({
        "doctype": "Workflow",
        "workflow_name": name,
        "document_type": document_type,
        "is_active": 1,
        "send_email_alert": 0,
        "workflow_state_field": "workflow_state",
        "states": [{"doctype": "Workflow Document State", **s} for s in states],
        "transitions": [{"doctype": "Workflow Transition", **t} for t in transitions],
    })
    wf.insert(ignore_permissions=True)
    print(f"  Created workflow: {name}")
    return wf.name
```

**Structure:**

1. `_discover_workflows()` -- Query all active Workflow documents: `frappe.get_all("Workflow", {"is_active": 1}, ["name", "document_type", "workflow_state_field"])`. Return list of workflow dicts with their states and transitions.

2. `_test_workflow_transitions(workflow_name, document_type)` -- For each workflow:
   a. Find or create a test document of the target doctype (use demo data from Plan 01). If no records exist, create a minimal test doc.
   b. Set the document to the workflow's initial state
   c. For each transition chain (Draft -> intermediate -> final):
      - Set user to a user with the transition's `allowed` role (use demo users from Plan 01: faculty1@nit.edu, registrar@nit.edu, finance@nit.edu, etc.)
      - Call `from frappe.model.workflow import apply_workflow; apply_workflow(doc, action_name)`
      - Verify `doc.workflow_state == expected_next_state`
      - If transition fails, record the error
   d. Return pass/fail for each transition tested

3. `_test_existing_workflows()` -- Specifically test the 3 pre-existing workflows:
   - Leave Application Workflow: Draft -> Pending HOD Approval -> Approved
   - Teaching Assignment Approval: Draft -> Pending Faculty -> Pending HOD -> Approved
   - Fee Refund Approval: Pending -> Pending Approval -> Approved

4. `_generate_report(results)` -- Write `WORKFLOW_AUDIT.md` to the phase directory (`.planning/phases/03.4-post-fork-backend-audit-fix/`) with:
   - Summary table: workflow name, doctype, states tested, transitions tested, pass/fail
   - Details of any failures
   - Total: X workflows, Y transitions tested, Z passed, W failed

5. `run()` -- Main entry:
   ```python
   def run():
       frappe.flags.ignore_permissions = True
       print("=== Workflow Audit ===")
       workflows = _discover_workflows()
       print(f"Found {len(workflows)} active workflows")
       results = []
       for wf in workflows:
           result = _test_workflow_transitions(wf["name"], wf["document_type"])
           results.append(result)
       _generate_report(results)
       # Summary
       passed = sum(1 for r in results if r["status"] == "pass")
       failed = sum(1 for r in results if r["status"] == "fail")
       print(f"Results: {passed} passed, {failed} failed out of {len(results)} workflows")
   ```

**Key details:**
- NEVER test with Administrator user (bypasses workflow checks per research pitfall). Always use role-specific demo users.
- After testing, restore `frappe.set_user("Administrator")` so subsequent operations work
- Wrap each workflow test in try/except so one failure doesn't stop the entire audit
- For doctypes that require mandatory fields, look up a minimal valid record from demo data
- Use `doc.reload()` before each transition to get fresh state from DB
- After apply_workflow on submittable doctypes where final state has doc_status=1, verify doc.docstatus==1

**D-10 compliance:** If any workflow fails, the script should print clear diagnostic info (workflow name, transition attempted, error message, current state vs expected state) so failures can be fixed immediately.
