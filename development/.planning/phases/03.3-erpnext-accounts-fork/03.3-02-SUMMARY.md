## Decisions Made
- **Stock functions as inline stubs:** Rather than just commenting out stock imports, replaced them with inline function definitions that return empty/zero values (e.g., `get_item_details` returns `frappe._dict()`, `get_item_tax_map` returns `{}`). - **temporary_flag as contextmanager:** Re-implemented `temporary_flag` (from erpnext.utilities.regional) as a Python contextmanager that manages frappe.local.flags directly, eliminating the erpnext.utilities dependency.
- **57 archived (not 56):** Found 4 additional POS/loyalty/subscription child tables (pos_customer_group, pos_item_group, loyalty_point_entry_redemption, subscription_invoice) that were missed in the original plan count.
- **sales_taxes_and_charges_template kept:** The template doctype (for configuring tax rules) was kept in active doctype/ while the child table (sales_taxes_and_charges) was correctly archived.
