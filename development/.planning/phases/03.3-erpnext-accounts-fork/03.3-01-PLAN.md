  - `^import erpnext$` -> `from university_erp.university_finance import _erpnext_compat as erpnext`
  - `^from erpnext import (.+)$` -> `from university_erp.university_finance._erpnext_compat import \1`
  - `from erpnext.exceptions` -> `from university_erp.university_finance._exceptions`
  - `from erpnext.controllers.` -> `from university_erp.university_finance.controllers.`

- `rewrite_all_imports()` — walks ALL `.py` files under `university_finance/` and calls `rewrite_imports_in_file` on each
- `stub_stock_imports()` — in `university_finance/utils.py`, replace the 3 stock import locations with the stub functions from `_erpnext_compat.py`
- `run_full_fork()` — orchestrates all steps in order: copy doctypes, copy reports, copy core files, update JSONs, rewrite imports, stub stock imports
- `if __name__ == "__main__": run_full_fork()` entry point

The script should use relative paths from `frappe-bench/apps/university_erp/` as the working directory. Source = `../erpnext/erpnext/accounts/`, destination = `university_erp/university_finance/`.
