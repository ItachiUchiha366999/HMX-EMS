## Decisions Made
- **Stock stubs return empty/zero:** `get_warehouse_account_map` returns `{}`, `get_stock_value_on` returns `0`, `is_perpetual_inventory_enabled` returns `False` -- university has no inventory
- **get_exchange_rate queries Currency Exchange doctype:** Falls back to 1.0 for single-currency setups
- **allow_regional is a no-op:** University does not use ERPNext's regional override system
- **Non-accounts erpnext imports stubbed as comments:** erpnext.setup/utilities/stock/buying/selling references become `# Stubbed: ...` to preserve code structure while breaking the dependency
- **Multi-line import continuation lines commented out:** When stubbing `from erpnext.X import (`, continuation lines also get `# ` prefix to avoid syntax errors
